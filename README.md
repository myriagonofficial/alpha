# Alpha
Amateur game developed by Myriagon


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Generate executables for Windows, Mac, Linux
```
npm run nw-build
```

### Lints and fixes files
```
npm run lint
```
